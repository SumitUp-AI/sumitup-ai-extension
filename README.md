# SumitUP AI Powered Meeting Assistant, Chrome Extension
SumitUp Chrome (Web) Extension for recording video and meeting, uploading to ASR Module to transcribe audio into text and later store in DB for transcribing data from it.

> # Warning
This extension is compatible with
| Chrome | MS Edge | Firefox | Safari |
|:-------: |:-------: |:--------: |:-----:|
| Yes, 100% | Yes, Edge supports Chrome Extensions | Partially, Doesn't work with Chrome Web APIs | No |
